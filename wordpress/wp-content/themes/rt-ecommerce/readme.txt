== Description ==

RT Ecommerce is a Free Ecommerce WordPress Theme. It can be used for any kind of stores like jewelry shop, online book market, cloth store, fashion store and much more. You can make any kind of store with this responsive Woocommerce Wordpress Theme. Our E-commerce Wordpress Theme is developed with the clean coding and most important by keeping all the WordPress coding standard in mind. You can set up your own store and start your business. You will get ample of features and customization option. The theme has a full-width banner which makes the user a great impact who comes to your WordPress website. It also has a call to action (CTA) which we believe a great way to achieve highest clicks which further help you in the lead generation. Our Ecommerce WordPress theme is SEO friendly which helps you to drive traffic to your website. You can showcase your featured products on the main page itself which attracts your visitors.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in RT Ecommerce in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.

==Front Page/Home page Customization ==
1. Go to Pages -> Add New. Add Title "Front page" select the template "Section Home" from the "Template" Drop down.
2. Publish the page.
3. Go to Settings -> Readings set the radio button to "Static Front Page" and then select the "Front page" you have created in the step 1.
4. Save the changes.

==Setup The Slider ==
1. Go to the Pages -> Add New. Add title "Slide 1", add featured image. Publish it.
2. Go to Appearance -> Customize -> RT Settings -> Slider Settings. Select the page you have created in step 1.
3. Repeat the same procedure for the other slides.

==Setup the Featured Products section ==
1. Go to plugins -> Add New -> serch "Woocommerce" -> Install -> Activate it.
2. Go to products -> Add New. Create products.
3. Go to pages -> Add New. Add title "featured Products" the add the shortcode to show the products like [products limit="4"]. Publish the page.
4. Go to Appearance -> Customize -> RT Settings -> Featured Products and set the page you have created in the step 3.

= How to change content or customize site? =
1. In the admin area, go to Appearance > Customize
2. You wll find different options in customizer
2. Add/Edit Widgets, Page and Post
